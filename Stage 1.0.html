Lesson 1: The Basics of Web and HTML

The World Wide Web

The World Wide Web is a collection of HTML Documents
HTML = HyperText Markup Language
Hyperlinks = Links between the pages
Many types of files are found on the web
1. Plain text
2. html
3. Images
4. Videos
5. Music
This Udacity video explains how the web works

Major Pieces of the Web

1) You - Your computer and the browser running on that computer
2) The Internet
3) Servers
Your browser makes a requests to the servers with a protocol called HTTP and the servers respond with files that the browsers displays
This Udacity video explains how the pieces fit and communicate with each other,

HTML Basics

HTML is made up of;
1) text content (What you see)
2) markup (what it looks like)
3) References to other documents (e.g images, videos, music)
4) links to other pages
This Udacity video explains what HTML is made up of

HTML Markups

HTML Markup is made of Tags
An element contains an Opening Tag, the content and the Closing Tag
Some examples of tags are the Bold tag and the Italics Tag
